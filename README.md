# 🔐 PeritoLab

Ferramenta profissional para Perícia Digital e Computação Forense.

O PeritoLab reúne um conjunto de utilitários desenvolvidos para auxiliar peritos judiciais, assistentes técnicos, advogados e profissionais de segurança da informação na preservação, análise e documentação de evidências digitais.

---

## 🌐 Demonstração

https://peritolab-production.up.railway.app/

---

# Recursos

✅ Gerador de Hash

- MD5
- SHA-1
- SHA-256
- SHA-512

---

✅ Verificador de Integridade

Comparação de hashes para validação de evidências digitais.

---

✅ Cadeia de Custódia

Geração de documentação para registro de coleta, preservação e movimentação de evidências.

---

✅ Metadata Inspector

Extração de metadados de arquivos:

- PDF
- DOCX
- JPG
- PNG

# Como executar

Clone o projeto

```bash
git clone https://github.com/lucianomonteirocaprara-hash/PeritoLab.git
```

Entre na pasta

```bash
cd PeritoLab
```

Execute

```bash
PORT=8080 ./start.sh
```

Acesse

```
http://localhost:8080
```
